Erfasst die Daten des Formulars über JavaScript. 

Das FormData-Objekt unterstützt denselben Mime-Typ wie ein HTML-Formular über 
JavaScript und bei Bedarf auch die Konvertierung in JSON.

Ruft die Fetch-API auf, um Daten an den Server zu senden

Verwendet nur die Elemente mit dem Attribut data-dynamic-form

Übergibt den Accept-Header, um den Server aufzufordern, im JSON-Format zu antworten.

Gibt an, ob die Antwort sowohl in Erfolgs- als auch in Fehlerszenarien verwendet wird.

Verwendet die Attribute data-success und data-error, um Daten aus dem Formular zu erhalten

Leert das Formular nach dem Absenden, um weitere Eingaben zu ermöglichen.

Löscht die Meldung nach xx Sekunden.
